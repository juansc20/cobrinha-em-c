# 🐍 Jogo da Cobrinha em C

Este é um jogo da cobrinha simples feito em linguagem C, para rodar no terminal do Windows. Desenvolvido como prática de programação básica e controle de fluxo.

## 🎮 Controles

- `W` - Cima  
- `S` - Baixo  
- `A` - Esquerda  
- `D` - Direita  
- `X` - Sair do jogo
